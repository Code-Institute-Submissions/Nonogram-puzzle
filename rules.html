HOW TO PLAY:

By switching the button shown here
you can switch between coloring or leaving the square.
Every number represents a sequence. for example:
5
This means all the 5 squares need to be colored.
If there is more than one number, there must be at least one blank square between each sequence. For example:
2 - 2
In this case there should be at least one black square between the sequences.
